# Mingle
> A Social Networking website to connect to your dear ones.

> Link to website : [shishirjindal.tk/mingle](http://shishirjindal.tk/mingle)

### Current Features
> Personal Chat

> Group Chat

> Follow your Friends

> Update Status

> Update your Profile Picture

> Block and Unblock specific people.
